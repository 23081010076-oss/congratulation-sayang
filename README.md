# 🎉 Landing Page Selamat Sempro - Ayu Komariyah

Landing page React yang lucu, menggemaskan, dan elegan untuk merayakan kesuksesan seminar proposal.

## ✨ Fitur

- 🎨 Animasi yang menggemaskan dan elegan
- 🎊 Efek confetti saat tombol ditekan
- 💖 Animasi hati yang terbang
- ✨ Sparkles dan elemen floating
- 🌈 Gradient background yang indah
- 📱 Responsive design
- 🎭 Animasi smooth dan interaktif

## 🚀 Cara Menjalankan

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Buka browser dan akses `http://localhost:5173`

## 🎯 Cara Menggunakan

- Klik tombol "Rayakan Kesuksesan!" untuk melihat efek confetti dan animasi hati
- Nikmati semua animasi lucu yang ada di halaman

## 🛠️ Built With

- React 18
- Vite
- CSS3 Animations
